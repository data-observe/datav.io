For enterprise applications, it is common to have a sidebar navigation that allows users to navigate between different sections of the application. 

Datav allow us to easily create our own sidebars in each team, and users can switch between these sidebars.

<img src={require('@/img/docs/admin/sidebar.jpg').default.src} alt="global-roles"  className="rounded-lg ring-1 ring-black/5"  style={{height: '500px'}}/>

In above image, the top section which the arrow points to be customized, and the bottom section is the default sidebar that can't be modifyied.

Because we have introduce this concept in [Sidebar tutorial](/docs/tutorial-sidebar), you can visit that page to learn more about sidebar.
