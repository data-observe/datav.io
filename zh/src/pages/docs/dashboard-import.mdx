在 [元数据](/docs/tutorial-team#dashboard-meta-data) 章节中，我们已经讲过什么是仪表盘元数据。因此，你应该已经了解该如何通过修改元数据来修改仪表盘。

但是这还不是全部，我们还可以使用元数据来完整的导入一个仪表盘，例如别人分享了一个好看的仪表盘，你可以导入其配置信息到自己的 xObserve 中。

## 复制元数据

打开 `Homne` 仪表盘的设置面板，选择 `元数据` 标签页，你将看到如下内容：

<img src={require('@/img/docs/dashboard/metadata.jpg').default.src} alt="metadata" width={3086} height={1658} className="rounded-lg ring-1 ring-black/5" />


二话不说，先拷贝这些 JSON 格式的元数据。

## 导入仪表盘

1. 点击侧边导航栏下半区域的 `+` 图标
2. 选择导入仪表盘标签页
3. 将刚才拷贝的元数据粘贴到 `Meta JSON` 中
4. 选择你想将仪表盘导入到的团队

<img src={require('@/img/docs/dashboard/import.jpg').default.src} alt="import" width={3086} height={1658} className="rounded-lg ring-1 ring-black/5" />


当然，在导入之前，我们还可以修改元数据，例如将标题修改为 <Code>  "title": "A imported dashboard" </Code>。

点击 `提交` 按钮，即可完成导入，非常简单。


## 哪些改变了？

当一个仪表盘被导入后，相比之前的仪表盘，到底哪些属性发生了变化？

答案就一个， dashboard id, 例如在上面的导入中，仪表盘 id 变化： <Code>d-home</Code> -> <Code>d-c52NkURSz</Code>.

因此，可以看出导入的和原始的是两个完全不同的仪表盘，虽然它们看起来一模一样。
