下面，我们来看看该如何使用自定义的背景图片：

1. 打开 `Home` 仪表盘的设置面板，选择 `样式` 标签页.
2. 不在为 `背景` 字段选择任何值，直接拷贝后面的字符串到输入框中 `https://github.com/xobserveio/assets/blob/main/dashboard/demo-background.jpg?raw=true` to input box
3. 按下 `Enter` 确认输入
4. 将 `背景主题` 设置为浅色 `Light`, 原因是我们的图片是浅色的
5. 启用背景

回到仪表盘页面，你会发现新的背景已经被应用：
