对于时间序列数据而言，最强大的就是 `Graph` 图表类型。

它的 x 轴永远是时间，y 轴是指标的值，因此 `Graph` 图表是时间序列数据的天生伴侣。

<img src={require('@/img/docs/plugin/graph.jpg').default.src} alt="graph" style={{height: '300px'}} className="rounded-lg ring-1 ring-black/5" />


下面一起来看看它的一些配，由于篇幅原因，这里只讲解较难理解的配置，其它的建议大家自行尝试。

## 连接空值 Connect null values 

当从数据源查询数据时，如果某些时间点没有数据，那么图表中就会出现无数据的部分：

<img src={require('@/img/docs/plugin/null-data.jpg').default.src} alt="null-data" style={{height: '300px'}} className="rounded-lg ring-1 ring-black/5" />


上图中，有大量的无数据部分，如果你将鼠标悬停在其中的一个空值上，你会发现它的 tooltip 中没有任何数据。

如果勾选了 `Connect null values` 选项，那么图表中的空值部分就会被连接起来：

<img src={require('@/img/docs/plugin/connect-null.jpg').default.src} alt="connect-null-data" style={{height: '300px'}} className="rounded-lg ring-1 ring-black/5" />
