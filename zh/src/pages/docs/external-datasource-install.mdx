安装数据源的外部插件跟安装图表插件的步骤几乎一模一样，唯一的区别就在于，下载下来的数据源插件需要放在 `ui/external-plugins/datasource` 下。

大家可以按照上一章节的方式安装一个数据源插件：demo，它是内置插件 Prometheus 的换皮。
